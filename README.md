## Project Description

This repository showcases my Digital Literacy Portfolio created as part of the Student Digital Ambassador initiative. It also includes a detailed project report along with supporting materials such as screenshots, email samples, and checklists.

## Project Structure

- README.md – Overview of the project, student details, and summary of all modules.
- Project_Report.pdf – Comprehensive report covering all five tasks.
- task-1-presentation/ – Infographic on digital literacy awareness with a short explanation.
- task-2-portfolio/ – Screenshots of digital profiles along with reflection notes.
- task-3-platforms/ – Coding platform activity, Google Form, and response screenshots.
- task-4-email-etiquette/ – Formal email drafts and a checklist of social media guidelines.
